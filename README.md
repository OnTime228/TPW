## TPW bot (это тпш, а не тпв...)

Тестовый кейс тпш

### Запуск приложения (~~да начнуться голодные игры~~):
1. Создайте `.env`:
- скопируйте `.env.example` → `.env`
- вставь `BOT_TOKEN`

2. Данные к данным:
- `./data/videos.json` (или ZIP с JSON и тогда `DATA_PATH=/data/your.zip`)

3. Запустите:
```bash
docker compose up --build```

4. Работает
